## Project setup 

### clone repository
``` 
git clone https://github.com/rishabhknowss/Echo-Serverless-blogging-site.git 
```
### start backend
```
cd server
npm install
npm run dev
```
### start frontend
```
cd client 
npm install 
npm run dev
```

### dependencies
 - node
 - cloudflare worker (wrangler)
